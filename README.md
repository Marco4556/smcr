# Salted Mayo Clicker: Resalted
A recreation of Salted Mayo Clicker in Pure Java

Salted Mayo Clicker: Resalted is a recreation of the Scratch game Salted Mayo Clicker (heyflame.itch.io/salted-mayo-clicker) written using Java and Java Swing. This is a work in progress, but you can still test it on your computer
